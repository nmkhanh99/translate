## 4 Activation Modes Chính Thức (theo Anthropic Agent Skills)

1. **Always On**  
   Áp dụng mọi lúc, mọi task.  
   Dùng khi: quy tắc phải luôn đúng.

2. **Manual**  
   Chỉ kích hoạt khi bạn gõ `@tên-rule` trong chat.

3. **Model Decision**  
   Agent tự quyết định dựa trên nội dung rule.

4. **Glob**  
   Chỉ áp dụng cho file khớp pattern.  
   Ví dụ: `Glob: src/**/*.py`
