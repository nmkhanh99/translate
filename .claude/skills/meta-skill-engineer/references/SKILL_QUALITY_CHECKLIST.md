## Skill Quality Checklist (theo Anthropic Agent Skills)

1. Description rõ ràng + chứa trigger chính xác?  
2. Có section Goal?  
3. When to use + Instructions step-by-step?  
4. Constraints + Best practices?  
5. Reference đúng relative path (scripts/references/examples)?  
6. Skill chỉ làm một việc duy nhất?  
7. Ngắn gọn (<150 dòng)?

Điểm: 7/7 = Perfect
