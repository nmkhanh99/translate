# Git Commit Formatter Skill

## When to use this skill
- User yêu cầu commit hoặc viết commit message.

## Instructions
1. Phân tích thay đổi (git diff).
2. Xác định type và scope.
3. Viết message theo chuẩn Conventional Commits.
